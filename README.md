# Pay-roll System for Icon-Africa

#About

  1. Calculates salaries for employees, using calculations taken from Icon's accountent.
  2. Generates payslips and print them.
  3. Save payslips for later use.
  
 #Technology used:
 
  1. Java
  2. Spring framework
  3. JDBC & MySQL.
  4. Vanilla JS.
  5. XML.
